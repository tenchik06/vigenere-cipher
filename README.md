# Vigenere Cipher

## Description
Implementation of the Vigenere cipher in Python with support for **Russian** and **English** alphabets.

## How it works
The Vigenere cipher uses a keyword to shift letters. Each letter of the text is shifted by a value corresponding to the keyword letter.

| Alphabet | Number of letters |
|----------|-------------------|
| Russian | 32 (А-Я, а-я) |
| English | 26 (A-Z, a-z) |

## Files
| File | Purpose |
|------|---------|
| `vigenere.py` | Main script |
| `input.txt` | Original text |
| `encrypted.txt` | Encrypted text |
| `decrypted.txt` | Decrypted text |
